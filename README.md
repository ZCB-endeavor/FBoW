# Fast Bag of Words
## Introduce
For visual-slam project FBoW.

## How to build

```bash
cd FBoW
mkdir build && cd build
cmake ..
make -j4
```

if you want to install this project, you can execute the following command.

```bash
sudo make install
```

This command will install the project under **/usr/local**
